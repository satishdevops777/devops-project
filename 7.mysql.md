## 1Ô∏è‚É£ Disable Default MySQL Module (MySQL 8)

### Why?

CentOS 8 enables MySQL 8 by default using AppStream modules. Installing MySQL 8 will break application compatibility.

Disable it:

```bash
dnf module disable mysql -y
```

---

## 2Ô∏è‚É£ Configure MySQL 5.7 Repository

### Why a custom repo?

MySQL 5.7 is **not available** in CentOS 8 default repositories.

We use the **official MySQL Community repository**.

---

### Create repo file

```bash
vim /etc/yum.repos.d/mysql.repo
```

### Repository configuration

```ini
[mysql]
name=MySQL 5.7 Community Server
baseurl=http://repo.mysql.com/yum/mysql-5.7-community/el/7/$basearch/
enabled=1
gpgcheck=0
```

> ‚ÑπÔ∏è This repository works across compatible RHEL-based systems.

---

## 3Ô∏è‚É£ Install MySQL 5.7 Server

```bash
dnf install mysql-community-server -y
```

Verify installation:

```bash
mysqld --version
```

Expected output:

```text
mysqld  Ver 5.7.x
```

---

## 4Ô∏è‚É£ Start and Enable MySQL Service

### Enable MySQL at boot

```bash
systemctl enable mysqld
```

### Start MySQL service

```bash
systemctl start mysqld
```

### Verify service status

```bash
systemctl status mysqld
```

Expected state:

```text
active (running)
```

---

## 5Ô∏è‚É£ Secure MySQL Installation

### Why secure installation?

By default:

* MySQL root account has a temporary or random password
* Weak security defaults are enabled

We must set a **known strong root password**.

---

### Set root password

```bash
mysql_secure_installation --set-root-pass RoboShop@1
```

> You may choose a different strong password as per your policy.

---

## 6Ô∏è‚É£ Validate MySQL Access

### Login using new password

```bash
mysql -uroot -pRoboShop@1
```

Expected result:

```text
Welcome to the MySQL monitor
```

This confirms:

* MySQL is running
* Root password is updated successfully

## üîê Security Notes (Production Perspective)

In real production environments:

* Root login is restricted
* Application uses a **separate DB user**
* Firewall limits access to MySQL port **3306**

RoboShop labs keep it simple for learning.
